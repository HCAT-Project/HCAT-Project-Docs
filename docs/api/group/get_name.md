# get_name

# 描述

获取群聊名

| 请求地址                | 请求方式     |
|---------------------|----------|
| /api/group/get_name | GET/POST |

| 参数       | 类型     | 描述   |
|----------|--------|------|
| group_id | String | 群聊ID |

| 返回参数       | 类型     | 描述    |
|------------|--------|-------|
| group_name | String | 群聊名   |
| remark     | String | 备注    |
| message    | String | 返回的信息 |
| status     | String | 状态    |
