# check

# 描述

检查文件是否存在

**!!\[饼干需要\]!!**

| 请求地址            | 请求方式 |
|-----------------|------|
| /api/file/check | POST |

| 参数        | 类型               | 描述            |
|-----------|------------------|---------------|
| auth_data | \[Cookie\]String | 存在\[饼干\]里面的数据 |

| 返回参数   | 类型     | 描述 |
|--------|--------|----|
| status | String | 状态 |
