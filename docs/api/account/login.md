# login

# 描述

登录账户

| 请求地址               | 请求方式     |
|--------------------|----------|
| /api/account/login | GET/POST |

| 参数       | 类型     | 描述   |
|----------|--------|------|
| user_id  | String | 用户ID |
| password | String | 密码   |

| 返回参数    | 类型     | 描述    |
|---------|--------|-------|
| message | String | 返回的信息 |
| status  | String | 状态    |
