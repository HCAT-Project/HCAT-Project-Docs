# rename

# 描述

重命名账户

**!!\[饼干需要\]!!**

| 请求地址                | 请求方式     |
|---------------------|----------|
| /api/account/rename | GET/POST |

| 参数        | 类型               | 描述            |
|-----------|------------------|---------------|
| name      | String           | 需要修改的用户名      |
| auth_data | \[Cookie\]String | 存在\[饼干\]里面的数据 |

| 返回参数    | 类型     | 描述    |
|---------|--------|-------|
| message | String | 返回的信息 |
| status  | String | 状态    |
