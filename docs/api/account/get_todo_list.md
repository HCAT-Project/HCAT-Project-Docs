# get_todo_list
# 描述

获取To-Do清单

**!!\[饼干需要\]!!**


| 请求地址 | 请求方式 |
| --- | --- |
| /api/account/get_todo_list | GET/POST |


|参数|类型|描述|
|---|---|---|
|auth_data|\[Cookie\]String|存在\[饼干\]里面的数据|


|返回参数|类型|描述|
|---|---|---|
|data|List|清单|
|message|String|返回的信息|
|status|String|状态|
