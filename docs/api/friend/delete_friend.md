# delete_friend
# 描述
删除好友

**!!\[饼干需要\]!!**

| 请求地址 | 请求方式 |
| --- | --- |
| /api/friend/delete_friend | GET |


|参数|类型|描述|
|---|---|---|
|friend_id|String|好友ID|
|auth_data|\[Cookie\]String|存在\[饼干\]里面的数据|

|返回参数|类型|描述|
|---|---|---|
|message|String|返回的信息|
|status|String|状态|
