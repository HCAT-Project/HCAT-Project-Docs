# rename
# 描述
重命名群聊

**!!\[饼干需要\]!!**

| 请求地址 | 请求方式 |
| --- | --- |
| /api/group/rename | GET |


|参数|类型|描述|
|---|---|---|
|group_id|String|群聊ID|
|name|String|新群名|
|auth_data|\[Cookie\]String|存在\[饼干\]里面的数据|

|返回参数|类型|
|---|---|---|
|message|String|返回的信息|
|status|String|状态|
