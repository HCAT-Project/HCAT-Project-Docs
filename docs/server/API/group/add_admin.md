# add_admin
# 描述

**!!\[饼干需要\]!!**

| 请求地址 | 请求方式 |
| --- | --- |
| /api/group/add_admin | GET |


|参数|类型|描述|
|---|---|---|
|group_id|String|群ID|
|member_id|String|成员ID|

|返回参数|类型|描述|
|---|---|---|
|message|String|返回的信息|
|status|String|状态|
